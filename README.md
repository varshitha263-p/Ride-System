# Ride-Sharing-System-Database-Project

This project implements a scalable relational database design for a ride-sharing platform, similar to Uber or Lyft. Designed for INFO 5707 (Data Modeling for Information Professionals), it encompasses entity modeling, business rules, user requirements, and SQL queries to support a robust web-based and mobile ride booking system.

Aims to develop a backend relational database to support core functionalities of a ride-sharing platform, including:

Real-time trip booking

Driver/passenger matching

Payment processing

Ratings and feedback

Historical ride tracking

Administrative monitoring


Technologies:

MySQL / PostgreSQL (Schema Design, Queries)

ER Diagrams(draw.io)

Normalization Techniques

Data Modeling Tools (Lucidchart, dbdiagram.io)
